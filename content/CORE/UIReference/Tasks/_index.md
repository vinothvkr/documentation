TrueNAS includes an easy to use interface for common tasks a sysadmin needs to preform on a NAS on a regular basis.  These can roughly be broken down into three groups.

 + System level tasks
   + Cron Jobs
   + Init and Shutdown scripts
   + S.M.A.R.T. tests
 + Data backup tasks
   + Rsync tasks
   + Cloud Sync tasks
 + ZFS tasks 
   + Snapshots
   + Resilvers
   + Scrubs
   + Replication
